> 基于《Iterative Research Pool for A-Shares and Southbound Hong Kong Stocks》报告及后续分析整理。  
> 目标：建立一个可迭代的行业与公司研究池，而不是一次性寻找几只买入股票。  
> 使用方式：用于未来 6—12 个月行业跟踪、公司研究卡维护、每日情报收集和关键词检索。  
> 说明：以下内容仅用于研究框架和信息跟踪，不构成投资建议。个股是否纳入实际研究池，仍需复核最新交易状态、财务数据、估值、港股通资格和个人风险承受能力。

# 1. 未来 1 年最应该关注的行业

根据报告，未来一年的研究重点应从“找股票”转为“跟踪行业景气变化 + 维护公司研究池”。核心主线是：

```text
科技制造和产业升级
→ 高现金流红利资产
→ 互联网和结构性消费
→ 资源周期战术性跟踪
→ 地产链和同质化产能过剩板块观察
```

报告强调，后续淘汰公司的核心变量应是：

```text
现金转换
营运资本质量
毛利率稳定性
杠杆
行业数据是否支持公司叙事
```

## 第一梯队：必须持续跟踪

### 1.1 AI 算力、半导体、电子硬件、光模块、PCB、先进封装、服务器、工业软件

这是未来 1 年最值得放在第一优先级的方向。

原因不是简单的“AI 概念热”，而是这一方向同时具备：

- 政策支持；
- 产业数据改善；
- 国产替代逻辑；
- 云厂商和企业资本开支验证；
- 港股通和南向资金对科技资产的持续关注。

根据报告中引用的官方数据，2026 年 5 月计算机、通信和其他电子设备制造业增加值同比增长 17.0%，1—5 月集成电路产量同比增长 25.4%；工业企业利润数据中，计算机、通信和其他电子设备制造业利润 1—5 月同比增长 103.9%。

这一方向不能只看“谁是 AI 概念股”，而要看谁真正拿到了订单，谁的收入能够转化成现金，谁没有出现应收账款和库存失控。

重点研究问题：

```text
AI 服务器订单是否真实？
光模块、PCB、服务器供应链是否出现订单和现金流同步改善？
半导体设备国产替代是否转化为收入和利润？
先进封装、Chiplet、HBM、CPO、硅光等是否形成可持续收入？
客户集中度是否过高？
出口限制是否影响订单和估值？
应收账款和存货是否过快增长？
```

报告也强调，AI、半导体、光模块、PCB、服务器链条要重点区分“订单驱动型公司”和“主题重估型公司”。

---

### 1.2 高端装备、机器人、工业自动化、航空航天、轨交、船舶

这条线对应的是：

```text
制造业升级 + 国产替代 + 资本开支验证
```

2026 年 1—5 月工业机器人产量同比增长 28.1%，5 月专用设备制造业增加值同比增长 9.1%，铁路、船舶、航空航天和其他运输设备制造业增加值同比增长 7.4%。这说明部分高端制造和装备方向仍具备较强景气度。

但这一方向也非常容易被概念炒高，因此关注点不能停留在“机器人”“低空经济”“航空航天”等标签，而应放在以下硬指标上：

```text
订单
合同负债
交付周期
毛利率
核心零部件自制率
客户结构
下游客户 CAPEX 是否真实扩张
ROIC 是否改善
存货是否异常增加
```

需要特别注意：机器人和低空经济都属于容易提前定价的主题，只有已经形成订单、收入、客户验证和现金流路径的公司，才适合进入核心研究名单。

---

### 1.3 电力系统、智能电网、特高压、储能、核电、燃气、公用事业

未来一年电力系统方向不应简单等同于“新能源发电公司”，而应重点研究：

```text
电网投资
新能源消纳
电力调度
储能经济性
核电建设
燃气顺价
公用事业现金流
```

这一方向有两种不同类型的机会：

1. **成长型设备公司**  
   包括电网自动化、特高压、柔直、变压器、储能系统、核电设备等。

2. **现金流型公用事业公司**  
   包括水电、核电、燃气、电力运营商等。

关键问题：

```text
国网、南网招标是否加速？
特高压项目是否核准？
储能中标价格是否止跌？
新能源消纳率是否改善？
电力现货市场和辅助服务机制是否改善收益？
核电机组是否持续核准和开工？
燃气公司能否顺价？
自由现金流是否覆盖资本开支和分红？
```

报告建议，这一方向重点看电网 CAPEX、招标份额、储能经济性、价格传导和自由现金流，而不是无差别买发电公司。

---

### 1.4 创新药、医疗器械、CXO、医疗服务、银发经济

医药不是整体买，而是分层研究。未来一年重点看三类公司：

```text
商业化能力强的器械龙头
已有临床或商业化验证的创新药平台
现金流和订单尚未明显恶化的 CXO 公司
```

医药的机会来自政策支持、老龄化、国产替代、创新药国际化和设备更新，但风险也非常高，包括：

```text
集采压价
医保谈判
研发失败
融资压力
现金消耗
研发费用资本化
海外合规和地缘风险
```

因此，医药研究不能只看管线故事，而应严格跟踪：

```text
CDE 临床进展
NMPA 批准
医保目录
商保创新药目录
license-out 交易
临床读数
商业化放量
现金 runway
```

报告提醒，医药方向不要急着“押科学赢家”，而应分成器械商业化龙头、现金较强的创新药平台和高风险临床主题池。

---

### 1.5 高股息央国企、电信运营商、能源、部分公用事业、部分银行保险

这条线应该成为研究池里的常驻防守板块。它的核心不是“股息率越高越好”，而是：

```text
自由现金流能否覆盖分红
资本开支是否可控
监管回报是否稳定
估值是否仍有安全边际
公司是否真正具备长期分红能力
```

未来一年这类资产适合承担“市场下行时的稳定器”角色，但要警惕高股息陷阱：

```text
利润下行
资本开支上升
分红不可持续
资源价格处于周期高位
监管回报下降
分红依赖举债
```

报告明确说，高股息篮子要长期保留，但研究重点不是表面股息率，而是分红安全、自由现金流覆盖、资产负债表、资本开支和估值是否已反映长期衰退。

## 第二梯队：选择性跟踪

### 1.6 平台互联网、本地生活、线上零售、游戏、旅游、新消费、品牌出海

消费总量并不强，所以不能全面乐观。报告指出，线上零售、服务零售、平台互联网、旅游、新消费和品牌出海仍有结构性机会，但必须选择性跟踪。

这一组只看两类公司：

1. **平台型现金流公司**  
   有回购、有利润率、有用户粘性、有广告或交易变现能力。

2. **品牌出海或结构性消费公司**  
   有渠道扩张、IP 能力、海外增长和毛利率保护。

重点研究问题：

```text
GMV 增长是否靠补贴？
MAU/DAU 是否稳定？
take rate 是否改善？
广告收入是否恢复？
履约成本是否下降？
利润率是否改善？
回购是否持续？
品牌出海是否真正贡献收入和利润？
```

如果 GMV 增长但利润率下降，说明增长质量不高。

---

### 1.7 有色、化工、煤炭、油气、航运、资源周期

资源周期未来一年值得跟踪，但必须放在“周期反转池”，不能用成长股逻辑。

报告指出，非铁金属冶炼压延、化工、煤炭等行业利润增速较高，但钢铁和非金属矿物利润明显下滑，说明周期内部差异很大。

资源股最重要的是四个变量同步：

```text
价格
库存
利润
现金流
```

涨价本身不是充分信号。真正的正面信号是：

```text
价格上涨能传导到利润
库存下降
经营现金流改善
公司没有在周期顶部激进扩产
分红政策保持纪律
资产负债表没有恶化
```

---

### 1.8 出海制造：汽车零部件、工程机械、家电、船舶、轨交

这类公司未来一年可以跟踪，但要非常警惕：

```text
出口强，利润弱
```

报告特别指出，汽车产量和增加值数据看起来不错，但行业利润承压，说明价格战会吞噬利润。

这一组重点看：

```text
海外收入占比
海外产能
关税风险
汇率
客户库存
应收账款
毛利率
售后服务能力
本地化生产能力
```

传统家装
传统钢铁

| 平台互联网 / 消费 | GMV、MAU/DAU、广告收入、take rate、履约成本、补贴强度、游戏版号、旅游数据、回购、用户留存 | 收入增长与利润率同步改善，回购持续，用户粘性高，补贴下降 | GMV 增长靠补贴，履约成本上升，监管风险变大，用户增长停滞 |
| 资源周期 | LME/SHFE 价格、库存、TC/RC、港口库存、化工价差、开工率、运价、矿山供给、资本开支 | 价格底部回升、库存下降、现金流转正、龙头扩市占但不盲目扩产 | 价格上涨但利润没改善，库存堆积，资产负债表变差，管理层高位扩产 |
| 出海制造 | 海外订单、关税、汇率、海运费、海外工厂利用率、客户库存、应收账款、毛利率 | 海外收入增长且毛利率稳定，本地化产能落地，大客户拓展 | 贸易摩擦升级，汇兑损失，客户去库存，海外扩张拖累现金流 |

建议把每个行业的状态分成三个标签：

```text
景气上行
景气验证中
景气证伪中
```

每天收集情报时，不要只记录新闻标题，而要判断它改变的是以下哪一个变量：

```text
需求
供给
价格
利润
现金流
估值
资金偏好
政策环境
```

# 8. 研究池维护结构

最终研究池建议长期保持三层结构：

```text
核心跟踪：30—50家公司
深度研究：10—20家公司
观察扩展：100—200家公司
```

如果后续连接完整财务数据库，可以进一步扩展为：

```text
长名单：300—800家公司
研究名单：80—150家公司
重点跟踪名单：30—50家公司
深度研究名单：10—20家公司
```

现阶段最值得先做研究卡的顺序：

```text
AI算力 / 半导体 / 光模块 / PCB
→ 高端装备 / 机器人 / 自动化
→ 电网 / 核电 / 公用事业
→ 创新药 / 医疗器械
→ 高股息央国企
→ 平台互联网 / 结构性消费
→ 资源周期
→ 出海制造
```

核心原则：

```text
先用行业数据判断方向，
再用公司财务和订单验证真伪，
最后才考虑估值和买点。
```

## 附：三类股票池的筛选指标

### A. 质量成长池

适合：科技制造、AI、半导体、机器人、高端装备、创新药中已经有商业化进展的公司。

核心指标：

```text
3年营收 CAGR
3年扣非净利润 CAGR
毛利率趋势
ROE / ROIC
研发费用率
经营现金流 / 净利润
存货周转天数
应收账款周转天数
合同负债或订单增速
客户集中度
海外收入占比
资本开支 / 收入
净现金或有息负债率
估值相对历史分位
```

重点不是“增长最快”，而是找：

```text
增长真实
现金流能跟上
竞争地位在增强
估值没有完全透支
```

---

### B. 稳健红利池

适合：央国企、公用事业、能源、电信、港股高股息 H 股、部分银行保险。

核心指标：

```text
股息率
分红率
过去5年分红稳定性
自由现金流 / 分红
净负债率
ROE 稳定性
资本开支周期
PB / PE / EV EBITDA
大股东增持或回购
监管定价风险
AH 溢价或折价
南向持股比例
```

重点不是“股息率越高越好”，而是找：

```text
分红有现金流支撑
估值低但不是价值陷阱
业务不会快速衰退
```

---

### C. 周期反转池

适合：有色、化工、资源、航运、部分机械、部分消费恢复链。

核心指标：

```text
行业价格指数
库存水平
产能利用率
行业利润同比
公司毛利率拐点
经营现金流拐点
资产减值是否结束
资本开支是否收缩
龙头市占率变化
PB 历史分位
```

重点是找：

```text
行业底部已经出现
公司资产负债表没有坏
价格弹性大于成本压力
管理层没有在周期高点盲目扩产
```

## 最终结论

未来一年，研究资源的优先级应为：

```text
1. AI算力 / 半导体 / 光模块 / PCB
2. 高端装备 / 机器人 / 自动化
3. 电网 / 核电 / 储能 / 公用事业
4. 创新药 / 医疗器械 / CXO
5. 高股息央国企 / 能源 / 电信 / 银行保险
6. 平台互联网 / 线上消费 / 品牌出海
7. 资源周期 / 有色 / 化工 / 油气 / 航运
8. 出海制造 / 汽车零部件 / 工程机械 / 家电 / 船舶 / 轨交
```

暂时后置：

```text
地产链
传统建材
传统钢铁
同质化光伏
锂电中游
纯价格战新能源整车
只有概念没有订单和现金流的主题股
```

最重要的原则：

```text
不要用新闻热度替代行业数据。
不要用营收增长替代现金流验证。
不要用高股息率替代分红安全。
不要用概念空间替代订单和利润。
不要用一次性筛选替代持续迭代。
```
